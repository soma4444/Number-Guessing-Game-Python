# Number Guessing Game (Python)

A simple console-based Number Guessing Game developed using Python. The program generates a random number between **1 and 50**, and the player has **7 attempts** to guess it correctly.

## Features

- 🎲 Random number generation
- 🔢 User guesses a number between 1 and 50
- 💡 Provides "Too High" or "Too Low" hints
- ✅ Input validation using exception handling
- ⏳ Limited to 7 attempts
- 🏆 Congratulates the player upon a correct guess
- ❌ Reveals the correct number if all attempts are used

## Technologies Used

- Python 3
- `random` module

## How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python guessing_game.py
```
